HTML、CSS、JS 基础展示

一个简单的网页，展示前端三大技术的基本功能

1. HTML - 页面结构

HTML (HyperText Markup Language) 用于定义网页的结构。

  • 使用标签来定义内容的类型
  • 标签通常成对出现，如 <div> 和 </div>
  • 包含头部、主体和页脚等基本结构
  • 支持文本、图片、链接、表格、表单等元素

2. CSS - 样式设计

CSS (Cascading Style Sheets) 用于美化网页的外观。

  • 设置颜色、字体、大小等样式
  • 控制元素的布局和定位
  • 支持响应式设计，适配不同设备
  • 包含动画和过渡效果

3. JavaScript - 交互功能

JavaScript 用于实现网页的交互效果和动态功能。

  • 处理用户事件（点击、鼠标移动等）
  • 操作 DOM 元素
  • 执行计算和逻辑操作
  • 实现动画和动态效果
